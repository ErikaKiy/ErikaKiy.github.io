1. Akar-akar Persamaan Nonlinear

Definisi :
Suatu akar dari suatu fungsi f(x) adalah nilai r sedemikian rupa sehingga nilai fungsi f(x) = 0, walaupun turunan fungsi itu tidak nol pada nilai r atau f'(r) ≠ 0

Teorema Ketunggalan Solusi
Apabila f(x) dapat diturunkan dan turunannya f'(x) memiliki tanda yang tetap dalam interval [a, b], maka dapat dipastikan bahwa hanya ada satu akar real dalam interval tersebut.

Contoh
Fungsi f(x) = x³ + 4x² âˆ’ 10 memiliki nilai f(1) = âˆ’5 dan f(2) = 14 sehingga ada akar real di antara [1, 2]. Lebih lanjut turunan f(x) adalah f'(x) = 3x² + 8x sehingga f'(1) = 11 dan f'(2) = 28. Karena tandanya tetap (sama-sama positif) maka hanya ada satu akar f(x) dalam interval [1, 2].

1. Metode Bisection
Metode bisection berbasis pada Teorema Nilai Tengah (intermediate value theorem) dan teorema eksistensi dengan Mencari titik tengah antara dua titik perkiraan, sehingga biasa disebut binary search method.

3. Metode Regula falsi
Metode regula falsi merupakan gabungan antara Bisection dan Secant yang memerlukan dua titik awal, a dan b sedemikian sehingga f(a).f(b) < 0. Metode ini menggunakan garis yang dibentuk oleh (a, f(a)) dengan (b, f(b)) yang dikaitkan dengan titik potong sumbu-x dan garis yang dibentuk kedua titik tersebut.

Sifat-sifat bilangan riil pada penjumlahan
Pada himpunan R dari bilangan-bilangan riil terdapat dua operasi biner yang dilambangkan dengan (+) dan (.) dimana masing-masing disebut penjumlahan dan perkalian. Operasi-operasi itu memenuhi sifat-sifat sebagai berikut:

Teorema I
(a) Jika z, a ∈ R sedemikian hingga z + a = a, maka z = 0.

Teorema II
(a) Jika a, b ∈ R, sedemikian hingga a + b = 0, maka b = -a.
(b) Jika a, b ∈ R dan a ≠ 0, sedemikian hingga ab = 1 maka b=1/b.

Misalkan a, b ∈ R sembarang. Maka (a) Persamaan a + x = b mempunyai penyelesaian tunggal x = (-a) + b.
(b) Jika a ≠ 0, persamaan a.x = b mempunyai penyelesaian tunggal x = 1/a.b.

Misalkan a ∈ R sembarang. Maka

Teorema V (Hasil penting dari sifat aljabar)
Misalkan a,b,c ∈ R sembarang. Maka
(a) Jika a ≠ 0, maka 1/a ≠ 0 dan 1/(1/a) = a.
(b) Jika a.b = a.c, a ≠ 0 maka b = c
(c) Jika a.b = 0, maka a = 0 atau b = 0
